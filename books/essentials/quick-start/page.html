The directions below show you how to build and run a sample Reach decentralized application (DApp).

Prerequisites

  1. Install GNU Make:

    1. $ make -v
    2. GNU Make 4.3

  2. Install Docker:

    1. $ docker -v
    2. Docker version 20.10.5

Install Reach

  1. Create a reach directory:

    1. $ mkdir ~/reach
    2. $ cd ~/reach

  2. Download the reach script to the reach directory:

    1. $ curl https://docs.reach.sh/reach -o reach ; chmod +x reach

  3. Verify that you can run the reach script:

    1. $ ./reach version
    2. reach 0.1

  4. Download Reach Docker images:

    1. $ ./reach update

Important

Many examples use reach [command] instead of ./reach [command] or ~/reach/reach [command]. You can do this by including ~/reach in your PATH variable. For example, add PATH=~/reach:$PATH to your startup file (e.g. .bashrc, .zshrc, etc.). Be sure to rerun your startup file by closing/opening your terminal or running source (e.g. source .zshrc). Once you do this, you can run reach from any current working directory.
