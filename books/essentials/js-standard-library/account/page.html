deploy

This method creates a Reach contract object, starts deployment of the contract based on backend code, and returns the contract without waiting for deployment to complete. To wait for deployment to complete, see getInfo.

Signature

  1. deploy(backend) => contract
  2. backend is a reference (e.g. build/index.main.mjs).
  3. contract is a Reach contract object.

Example Program

  1. import { loadStdlib } from '@reach-sh/stdlib';
  2. import * as backend from './build/index.main.mjs';
  3. (async () => {
  4.   const stdlib = await loadStdlib();
  5.   const balance = stdlib.parseCurrency(10);
  6.   const account = await stdlib.newTestAccount(balance);
  7.   const contract = account.deploy(backend);
  8.   console.log(contract)
  9. })();
