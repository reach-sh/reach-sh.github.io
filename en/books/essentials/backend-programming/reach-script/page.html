The reach command-line script is the primary tool for Reach developers:

  1. $ reach help
  2. reach 0.1.6 (66f7fd96) - Reach command-line tool
  4. Usage: reach COMMAND
  6. Available options:
  7.   -h,--help                Show this help text
  9. Available commands:
  10.   config                   Configure default Reach settings
  11.   compile                  Compile an app
  12.   clean                    Delete 'build/$MODULE.$IDENT.mjs'
  13.   init                     Set up source files for a simple app in the current
  14.                            directory
  15.   run                      Run a simple app
  16.   down                     Halt all Dockerized Reach services and devnets
  17.   scaffold                 Set up Docker scaffolding for a simple app in the
  18.                            current directory
  19.   react                    Run a simple React app
  20.   rpc-server               Run a simple Reach RPC server
  21.   rpc-run                  Run an RPC server + frontend with development
  22.                            configuration
  23.   devnet                   Run only the devnet
  24.   upgrade                  Upgrade Reach
  25.   update                   Update Reach Docker images
  26.   docker-reset             Kill and remove all Docker containers
  27.   version                  Display version
  28.   hashes                   Display git hashes used to build each Docker image
  29.   help                     Show usage

reach clean

  1. $ reach clean -h
  2. Usage: reach clean [MODULE] [IDENT]
  3.   Delete 'build/$MODULE.$IDENT.mjs'
  5. Available options:
  6.   -h,--help                Show this help text
  8. MODULE is "index" by default
  9. IDENT  is "main"  by default
  11. If:
  12.  * MODULE is a directory then `cd $MODULE && rm -f "build/index.$IDENT.mjs";
  13.  * MODULE is  then `rm -f "build/$MODULE.$IDENT.mjs"

  2. Usage: reach compile [-o|--output DIR] [SOURCE] [EXPORTS...] 
  3.                      [--intermediate-files] [--install-pkgs] [--stop-after-eval]
  4.                      [--verify-timeout TIMEOUT-MS]
  5.   Compile an app
  7. Available options:
  8.   -o,--output DIR          Directory for output files
  9.   --intermediate-files     Store intermediate files in output DIR
  10.   --install-pkgs           Allow Reach to fetch remote package imports
