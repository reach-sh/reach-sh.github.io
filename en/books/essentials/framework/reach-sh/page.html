1. $ reach help
  2. Usage: reach COMMAND
  3.  where COMMAND is one of
  4.   compile      --- compile an app
  5.   clean        --- delete compiled artifacts
  6.   init         --- set up source files for a simple app
  7.   run          --- run a simple app
  8.   down         --- halt any Dockerized devnets for this app
  9.   scaffold     --- set up Docker scaffolding for a simple app
  10.   react        --- run a simple react app
  11.   rpc-server   --- run a simple Reach RPC server
  12.   rpc-run      --- run an RPC server + frontend with development configuration
  13.   devnet       --- run only the devnet
  14.   upgrade      --- upgrade Reach
  15.   update       --- update Reach Docker images
  16.   docker-reset --- docker kill and rm all containers
  17.   version      --- display version
  18.   hashes       --- display git hashes used to build each Docker image
  19.   help         --- show this info
