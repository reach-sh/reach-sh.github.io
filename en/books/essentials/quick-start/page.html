This page shows you how to run a Reach Hello World decentralized application (DApp) in a terminal:

Set up your environment

Reach assumes the presence of a POSIX-compliant shell and Docker.

MacOS and Linux

  1. Install GNU Make or verify your installation:

    1. $ make -v
    2. GNU Make 4.3

  2. Install Docker or verify your installation:

    1. $ docker -v
    2. Docker version 20.10.5

Windows

This video demonstrates the steps below.

  1. Install Windows 10, version 2004 or higher.

  2. Enable the Windows Subsystem for Linux feature, specifically the WSL 2 option.

  3. Download and install Docker Desktop, and enable the Docker Desktop WSL 2 backend.

  4. Follow the instructions from Docker on developing with Docker and WSL 2.

Install Reach

  1. Create a reach directory:

    1. $ mkdir ~/reach
    2. $ cd ~/reach

  2. Download the reach script to the reach directory:

    1. $ curl https://docs.reach.sh/reach -o reach ; chmod +x reach

  3. Verify that you can run the reach script:

    1. $ ./reach version
    2. reach 0.1

  4. Download Reach Docker images:

    1. $ ./reach update

Create the app

  1. Create a project subfolder (as a child of ~/reach):

    1. $ mkdir hello-world
    2. $ cd hello-world

  2. Create source files:

    1. $ ~/reach/reach init
    2. Writing index.rsh...
    3. Writing index.mjs...
    4. Done.

    index.mjs is the JS frontend, and index.rsh is the Reach backend. We defer code analysis until the Tutorials.

Run the app

  1. Run the DApp on a local dockerized Ethereum devnet:

    1. $ ~/reach/reach run
    2. ...
    3. Hello, Alice and Bob!
    4. Launching...
    5. Starting backends...
    6. Goodbye, Alice and Bob!

    You could also use REACH_CONNECTOR_MODE=ETH-devnet ~/reach/reach run.

  2. Run the DApp on a local dockerized Algorand devnet:

    1. $ REACH_CONNECTOR_MODE=ALGO-devnet ~/reach/reach run
    2. ...
    3. Hello, Alice and Bob!
    4. Launching...
    5. Starting backends...
    6. Goodbye, Alice and Bob!

  3. Run the DApp on a local dockerized Conflux devnet:

    1. $ REACH_CONNECTOR_MODE=CFX-devnet ~/reach/reach run
    2. ...
    3. Hello, Alice and Bob!
    4. Launching...
    5. Starting backends...
    6. Goodbye, Alice and Bob!
